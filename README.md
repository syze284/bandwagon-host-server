**画像**：第一次买 VPS，不确定自己真正需要什么，想先试试水，钱不想花太多。

这类用户其实最痛苦——不知道选什么，又怕选错了浪费钱。

搬瓦工针对这个人群的答案很清楚：**CN2 GT 套餐（KVM 入门款），$49.99/年起**。

这是目前搬瓦工常规在售套餐里最便宜的一档。1核2GB内存（高配档位），最小档1G内存，20G SSD，1TB月流量，默认洛杉矶机房，购买后可后台一键切换到 DC3 CN2、DC8 ZNET 等共9个机房，其中 DC3 CN2 机房走电信 CN2 GT 线路，对新手来说体验不差。

对新手来说，这个套餐的价值不在于"最好"，而在于"买错了不心疼"，先上手，再升级。
